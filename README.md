使用nodejs koa2 mongodb 搭建 api 管理

使用微信扫码免注册登录  

使用vue iview iview-admin 搭建前端管理页面

使用支付宝 实现收款 

使用websocket 实现用户登录 支付成功 推送通知

使用aliyun oss+cdn 实现静态资源缓存 客户端下载
